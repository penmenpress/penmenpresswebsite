PDF extension for WP ERP

== Description ==
WP ERP PDF extension adds the ability to export any payment or invoice from WP ERP in PDF file.



== Installation ==

= Manual Installation =

1. Upload the entire `/erp-pdf` directory to the `/wp-content/plugins/` directory.
2. Activate WP ERP - PDF Invoice through the 'Plugins' menu in WordPress.
